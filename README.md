# 🏰 Tiny Tower Defense

Welcome, brave commander.

The enemy army is coming...\
Your towers are ready...\
Your strategy is questionable... but let's try anyway.

This is a **simple HTML Tower Defense game** where your only job is:

-   Build towers 🗼\
-   Stop enemies 🐉\
-   Protect your base ❤️\
-   Panic when everything goes wrong 😅

If the enemies break through... well... let's just say the kingdom will
remember you.

------------------------------------------------------------------------

## 🎮 Play the Game

Clone the repository and run the game locally.

``` bash
git clone https://github.com/yourusername/tower-defense-game
cd tower-defense-game
```

Then open:

    index.html

in your browser.

------------------------------------------------------------------------

## 🧠 Strategy Tips

-   Place towers wisely
-   Don't waste money early
-   Enemies get stronger each wave
-   Sometimes panic‑building towers everywhere also works

------------------------------------------------------------------------

## ⚔️ Mission

Survive as many waves as possible.

Good luck commander.\
The kingdom is counting on you.
